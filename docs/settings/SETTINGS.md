# 설정 시스템

| 항목 | 배점 | 통과 조건 |
| --- | ---: | --- |
| 핵심 기능 | 35 | 사운드, 화면, 대화, 접근성 설정과 기본값 복원 제공 |
| 조작과 피드백 | 25 | 마우스/키보드 조작, 즉시 미리보기, 저장 및 오류 상태 안내 |
| 저장 안정성 | 20 | 새로고침 후 복원, 손상되거나 범위를 벗어난 저장값 복구 |
| 접근성 | 15 | 의미 있는 레이블, 키보드 포커스, 상태 알림, 효과 감소 지원 |
| 코드와 검증 | 5 | UI와 저장 로직 분리, 자동 테스트 통과 |

## 구현 범위

- 사운드: 전체/BGM/효과음 음량, 채널별 스피커 버튼 음소거
- 화면: 창/전체 화면, 흔들림·깜빡임·빠른 전환 효과 줄이기
- 대화: 클릭/자동 진행, 느림/중간/빠름 글자 출력 속도, 자동 진행 대기시간
- 가독성: 글자 크기, 대화창 투명도, 읽은 대사만 건너뛰기
- 공통: 변경 즉시 저장, 기본값 복원, 대화창 미리보기

설정값은 `nan-game-settings-v1` 키로 `localStorage`에 저장한다. 실제 오디오가 추가되면
`data-master-muted`, `data-bgm-muted`, `data-sfx-muted`와 `--master-volume`,
`--bgm-volume`, `--sfx-volume` 값을 오디오 매니저의
Master/BGM/SFX 버스에 연결한다.

## 자체 평가와 개선 기록

1차 구현은 자동 테스트를 통과했지만 실제 브라우저에서 설정 창을 열 때 라디오 그룹에
슬라이더 출력 처리 로직이 적용되어 예외가 발생했다. 출력 갱신 대상을 실제 폼 요소로
제한한 뒤 다시 검증했다.

최종 결과: **97/100**

- 핵심 기능 34/35: 오디오 파일이 아직 없어 음량값은 저장 및 연동 인터페이스까지만 제공
- 조작과 피드백 25/25
- 저장 안정성 20/20
- 접근성 13/15: 데스크톱 브라우저 검증 완료, 별도 스크린 리더 실기기 검증은 미실시
- 코드와 검증 5/5

브라우저에서 설정 열기, 값 변경, 완료, 새로고침 후 복원, 기본값 복원을 확인한다.

## 의존성 및 권한 주의사항

- 외부 패키지 의존성은 없다.
- 전체 화면 전환은 브라우저 정책상 사용자 입력 중에만 요청할 수 있으며 권한이 거절될 수 있다.
  실패하면 실제 화면 상태로 설정값을 되돌리고 설정 하단에 오류를 표시한다.
- 브라우저 캐시 때문에 수정한 정적 파일이 즉시 반영되지 않을 수 있어 HTML의 CSS/JS URL에
  버전 쿼리를 사용한다. 파일을 변경해 배포할 때 버전 값도 올린다.
