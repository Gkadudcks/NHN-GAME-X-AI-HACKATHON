# 비주얼 스타일 가이드

## 고정 스타일

- 장르: 현대 한국 오피스 로맨스 비주얼 노벨, 가벼운 미스터리 요소
- 표현: 정돈된 상업용 애니메이션 비주얼 노벨 일러스트
- 선화: 얇고 깨끗하며 굵기가 일관된 선
- 명암: 부드러운 2~3단 셀 셰이딩, 붓 자국과 과도한 질감 없음
- 인체: 성인의 자연스럽고 현실적인 비율, 과장된 모에/치비 비율 금지
- 팔레트: 낮거나 중간 채도, 부드러운 대비, 네이비·베이지·크림·뮤트 브라운·페일 블루 중심
- 조명: 따뜻한 실내 확산광과 약한 차가운 환경 그림자
- 마감: 깨끗한 외곽, 게임 합성에 적합한 형태, 읽기 쉬운 실루엣

공통 스타일의 실제 생성용 영문 블록은 `assets/art/prompts/templates/common_style.txt`를 사용합니다. 프로젝트 전체에서 임의로 다시 쓰지 말고, 변경 시 `prompt_version`을 올립니다.

## 금지 방향

- 실사, 3D CGI, 유화, 수채화, 스케치 선화
- 두꺼운 외곽선, 극단적 애니 비율, 큰 눈, 치비 비율
- 한국 웹툰식 반실사 렌더링
- 네온/사이버펑크 팔레트, 높은 채도, 과도한 블룸
- 요청되지 않은 극적인 림 라이트, 강한 심도 효과
- 읽을 수 있는 텍스트, 로고, 워터마크, 임의의 배경 인물

네거티브 템플릿은 실패 유형 중심으로 짧게 유지합니다. 서로 충돌하는 스타일 키워드를 나열하지 않습니다.

## 변경 관리

바뀌면 안 되는 항목은 공통 스타일, 캐릭터 정체성, 표준 캔버스/구도, 장소의 구조입니다. 장면별로 바꿀 수 있는 항목은 승인된 표정·pose, 손동작/소품, 시간대와 조명 변형, 카메라 프레이밍입니다. 모델 변경은 스타일 변경으로 간주하며, 기존 마스터와 비교 검수 없이는 프로젝트 중간에 교체하지 않습니다.
