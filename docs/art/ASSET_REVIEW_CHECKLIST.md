# 이미지 에셋 검수 체크리스트

## 모든 에셋

- [ ] 기존 에셋 재사용으로 해결할 수 없는가?
- [ ] 공통 스타일, 채도, 대비, 선화, 명암 단계가 승인본과 일치하는가?
- [ ] 요청하지 않은 텍스트, 로고, 워터마크, 인물, 액세서리가 없는가?
- [ ] 손, 눈, 치아, 귀, 의복 경계, 반복 무늬에 생성 오류가 없는가?
- [ ] 파일명과 `vNNN`이 규칙에 맞고 기존 승인 파일을 덮어쓰지 않았는가?
- [ ] generation log에 모델/버전, 프롬프트 버전, 참조, 강도, seed, 해상도, 설정이 있는가?
- [ ] manifest 경로, 상태, generation log ID, `active_version`이 맞는가?

## 캐릭터 스프라이트

- [ ] 얼굴형, 눈 모양/색, 머리 길이/가르마/컬이 마스터와 같은가?
- [ ] 체형, 키 비율, 의상, 사원증, 귀걸이와 시계가 명세와 같은가?
- [ ] pose와 expression이 허용 목록에 있는가?
- [ ] 1024x1536 RGBA PNG이며 실제 투명 영역이 있는가?
- [ ] 눈·허리·발 기준선과 인물 크기가 다른 승인 스프라이트와 같은가?

## 배경

- [ ] 1920x1080 RGB 이미지이며 알파 채널이 없는가?
- [ ] 기준 장소의 문, 창, 책상과 소품 배치가 유지되는가?
- [ ] 캐릭터/UI를 위한 빈 공간과 가독성이 충분한가?
- [ ] 시간대 변형이 구조 재생성이 아닌 일관된 편집으로 보이는가?

## 이벤트 CG 및 최종 승인

- [ ] 1920x1080 RGB 이미지이며 알파 채널이 없는가?
- [ ] 리뷰 담당자와 승인 시간이 manifest에 기록되었는가?
