# 캐릭터 에셋 가이드

## 고정 정보

캐릭터별 `CHARACTER_SPEC.md`에는 얼굴형, 눈 모양/색, 코와 입, 헤어 길이/가르마/컬, 체형과 키, 기본 의상, 고정 액세서리, 시각적 성격을 구체적으로 기록합니다. 생성 시 지정 마스터 참조를 우선하며, 장면 프롬프트는 명시된 변경분만 다룹니다.

기존 캐릭터의 얼굴, 헤어, 체형, 의상, 액세서리를 생성 모델이 임의로 재해석하게 두지 않습니다. 손가락, 눈동자 방향, 귀걸이 쌍, 손목시계 위치, 옷깃과 사원증도 검수 대상입니다.

## 표준 스프라이트

- 캔버스: 1024 x 1536 px, 세로 2:3
- 형식: RGBA PNG, 투명 배경
- 카메라: 눈높이, 정면에 가까운 3/4, 자연스러운 50~70mm 느낌
- 전신 기준: 머리 위 여백 약 5%, 눈 20~23%, 허리 52~55%, 발 94~96%
- 기본 pose 하나를 먼저 승인하고 표정 레이어/변형 재사용을 우선한다.

프로젝트가 다른 규격으로 전환될 때는 manifest의 `character_sprite` profile과 이 문서를 함께 바꾸고 전 에셋을 재검증합니다.

## 제한된 장면 값

허용 값의 원본은 `art-assets.json`의 `controlled_values`입니다.

- pose: `neutral_standing`, `relaxed_standing`, `arms_folded`, `holding_cup`
- expression: `neutral`, `gentle_smile`, `embarrassed`, `surprised`, `concerned`, `tired`
- position: `left`, `center`, `right`
- framing: `full_body`, `waist_up`

`position`은 같은 이미지를 화면 어디에 배치할지 나타내므로 이미지 ID나 파일명에 넣지 않습니다. 새 값은 시나리오에서 즉석 생성하지 않습니다. 기존 값으로 연출할 수 없는 장면이 반복되는지 확인하고, 아트/시나리오 담당자가 manifest 목록과 문서를 함께 수정한 뒤 사용합니다.

## 재사용 우선순위

1. 같은 몸통 + 기존 표정
2. 같은 몸통 + 새 표정 레이어
3. 승인 pose + 작은 손/소품 변형
4. 새 pose
5. 완전히 새로운 전신 생성

마지막 두 단계는 일정과 일관성 비용을 검토한 경우에만 선택합니다.
