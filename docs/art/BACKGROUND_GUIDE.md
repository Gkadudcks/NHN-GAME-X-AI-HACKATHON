# 배경 에셋 가이드

## 분리 원칙

배경에는 캐릭터를 합성하지 않습니다. 캐릭터 스프라이트와 독립적으로 만들며, 인물·읽을 수 있는 글자·회사 로고를 넣지 않습니다. 전경에는 캐릭터와 UI를 배치할 여백을 둡니다.

## 표준 규격과 스타일

- 캔버스: 1920 x 1080 px, 16:9
- 형식: RGB PNG 또는 JPEG, 알파 채널 없음
- 카메라: 눈높이, 35mm 정도의 자연스러운 원근
- 표현: 깨끗한 애니메이션 비주얼 노벨 배경, 부드러운 셀 페인팅, 중간 디테일
- 팔레트: 네이비·회색·베이지·목재색, 따뜻한 사무실 조명
- 구조: 문, 창문, 책상, 통로와 소품의 위치가 버전 간 유지되어야 함

## 장소 재사용과 변형

장소 하나를 먼저 기준 배경으로 승인한 뒤 `day`, `evening`, `night`, `incident` 같은 시간/사건 변형은 구조를 다시 생성하지 않고 색보정과 부분 편집으로 만듭니다. 새 장소를 추가하기 전에 기존 장소의 크롭, 조명 변형, 소품 레이어로 해결 가능한지 확인합니다.

안정 ID 예시는 `background.office.day`입니다. 시간대는 ID의 마지막 의미 요소로 사용하되 파일 버전은 별도의 `vNNN`으로 관리합니다.
